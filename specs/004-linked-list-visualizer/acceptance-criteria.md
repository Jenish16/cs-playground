# 004 — Linked List Visualizer: Acceptance Criteria

- [ ] Nodes render connected by directional arrows; head is labeled; tail shows null.
- [ ] Insert at head adds a node before the current head and rewires the arrow.
- [ ] Insert at tail / after index places the node correctly.
- [ ] Delete at index removes the node and re-points the predecessor's arrow.
- [ ] Search/traverse animates a cursor moving node by node.
- [ ] Search reports found (with highlight) or not-found via message.
- [ ] Invalid index / empty list shows a message and changes nothing.
- [ ] The complexity note matches the selected operation.
- [ ] `npm run lint` passes.
